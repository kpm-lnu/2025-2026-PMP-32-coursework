# Побудова маршруту польоту дрона з урахуванням перешкод на основі задачі комівояжера


## Необхідні бібліотеки

- Python 3.9 або новіший
- [Webots](https://cyberbotics.com/) - для запуску симуляції

## Запуск

### 1. Порівняння евристичних алгоритмів для задачі комівояжера

Порівняння найближчого сусіда, найближчого вставлення та мурашиної колонії
на випадкових 3D-точках для N = 10, 20, 30, 40, 50, 60:

```bash
python3 methods/test.py
```

Як результат отримаємо таблицю з довжиною туру і часом для кожного алгоритму.

### 2. Симуляція в Webots

1. Відкрити Webots -> **File -> Open World** -> вибрати `webots/worlds/tsp.wbt`.
2. Натиснути Play.
3. Контролер `tsp_controller.py` запуститься автоматично і:
   - додасть на сцену 63 перешкоди з `webotsvs/obstacles.py`;
   - побудує граф (waypoint-и + допоміжні точки);
   - порахує матрицю енергетичних відстаней через Дейкстру;
   - знайде тур мурашиним алгоритмом;
   - проведе дрона по маршруту і виведе підсумкову таблицю у консоль.
