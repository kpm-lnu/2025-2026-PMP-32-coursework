## 📁 Структура репозиторію

```
📦 репозиторій/
├── дод_А.py              # Головний скрипт: навчання моделі, графіки, confusion matrix
├── дод_Б2.py             # Скрипт побудови порівняльних графіків по експериментах
├── дод_В1.py             # Скрипт оцінки моделі та побудови confusion matrix
├── README.md             # Цей файл
│
└── logs/                 # Папка з CSV-логами експериментів (створити вручну)
    ├── exp1_depth/
    │   └── model_2blocks.csv
    ├── exp2_hparams/
    │   └── lr_0_001_bs_64.csv
    ├── exp3_regularization/
    │   └── dropout_0_5.csv
    ├── exp4_augmentation/
    │   └── basic_aug.csv
    └── exp5_transfer/
        └── transfer_learning.csv
```

**Файли, що створюються автоматично після запуску `дод_А.py`:**
```
training_log.csv        — лог метрик навчання по епохах
training_loss.png       — графік функції втрат (Loss)
training_accuracy.png   — графік точності (Accuracy)
confusion_matrix.png    — матриця помилок
example_0.png           — приклад класифікації №1
example_1.png           — приклад класифікації №2
example_2.png           — приклад класифікації №3
example_3.png           — приклад класифікації №4
example_4.png           — приклад класифікації №5
best_model.pth          — збережена найкраща модель
```

## 🚀 Встановлення та запуск

### Крок 1 — Встановлення Python

Завантажити Python 3.10 з офіційного сайту:  
👉 https://www.python.org/downloads

> ⚠️ При встановленні обов'язково поставте галочку **"Add Python to PATH"**

---

### Крок 2 — Встановлення бібліотек

Відкрийте командний рядок (cmd) і виконайте:

```bash
pip install torch torchvision matplotlib numpy seaborn scikit-learn pandas
```

> ⏱️ Встановлення займає 5–10 хвилин залежно від швидкості інтернету.

---

### Крок 3 — Підготовка файлів

Розмістіть у **одній папці** наступні файли:
- `дод_А.py`
- `дод_Б2.py`
- `дод_В1.py`

---

### Крок 4 — Запуск основного скрипту

Переконайтесь що є підключення до Інтернету (датасет CIFAR-10 завантажується автоматично при першому запуску — ~170 МБ).

```bash
python дод_А.py
```


## 📊 Підготовка CSV-логів для порівняльних графіків (дод_Б2.py)

Для побудови порівняльних графіків усіх 5 експериментів потрібно створити папку `logs/` і вручну додати CSV-файли з даними таблиць Додатку Б.3.

### Структура папок

Створіть папки:
```
logs/
  exp1_depth/
  exp2_hparams/
  exp3_regularization/
  exp4_augmentation/
  exp5_transfer/
```

### Формат кожного CSV-файлу

Перший рядок — заголовок, далі дані по епохах:
```
epoch,train_loss,val_loss,train_acc,val_acc
```
