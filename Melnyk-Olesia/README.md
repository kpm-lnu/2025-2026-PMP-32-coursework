# Чисельне моделювання просторового поширення інфекції методом скінченних елементів

## Опис проєкту

У проєкті реалізовано чисельне моделювання просторового поширення інфекції методом скінченних елементів (FEM).

Розглядається рівняння:

-Δu + u = f(x, y)

де:
- u(x, y) - концентрація інфекції;
- f(x, y) - функція джерела інфекції.

## Реалізовані можливості

- побудова трикутної FEM-сітки;
- формування локальних та глобальних матриць;
- накладання граничних умов Діріхле;
- чисельний розв’язок системи рівнянь;
- візуалізація результатів.

## Проведені експерименти

У програмі досліджено декілька типів джерел інфекції:
- центральне джерело;
- два осередки інфекції;
- лінійне джерело;
- синусоїдальне джерело.
