# Відстеження рухомого об'єкта за допомогою Particle Filter

Цей проєкт реалізує систему відстеження рухомого об'єкта у відео за допомогою Particle Filter (фільтра частинок) та алгоритму віднімання фону з використанням Python та OpenCV.

Програма:

- виконує детекцію рухомого об'єкта;
- прогнозує рух об'єкта за допомогою частинок;
- оцінює положення об'єкта;
- формує область пошуку;
- будує траєкторію руху;
- візуалізує процес відстеження.

---

# Можливості

- Віднімання фону (Background Subtraction)
- Виявлення контурів рухомих об'єктів
- Particle Filter для відстеження
- Прогнозування положення об'єкта
- Оцінка стану на основі ваг частинок
- Перевибірка частинок (Resampling)
- Побудова траєкторії руху
- Візуалізація області пошуку
- Відображення маски руху

# Принцип роботи алгоритму

## 1. Віднімання фону

Для знаходження рухомого об'єкта використовується алгоритм:

```python
cv2.createBackgroundSubtractorMOG2()
```

Алгоритм формує маску переднього плану та виділяє області руху.

---

## 2. Морфологічна обробка

До маски застосовуються:

- Morphological Opening
- Dilate

для усунення шумів та заповнення розривів.

---

## 3. Детекція об'єкта

Програма:

- знаходить контури;
- вибирає найбільший контур;
- обчислює центр об'єкта.

---

## 4. Particle Filter

Кожна частинка містить:

- координати `(x, y)`
- швидкість `(vx, vy)`

Фільтр виконує:

1. прогнозування руху;
2. оновлення ваг;
3. оцінку стану;
4. перевибірку частинок.

---

## 5. Прогнозування стану

Для прогнозування використовується модель руху:

\[
x' = x + v \cdot dt + noise
\]

де:
- \(x\) — координати;
- \(v\) — швидкість;
- \(dt\) — часовий крок;
- \(noise\) — випадковий шум.

---

## 6. Оновлення ваг частинок

Ваги оновлюються відповідно до відстані між частинкою та вимірюванням:

\[
w = e^{-\frac{||x-z||^2}{2R}}
\]

де:
- \(x\) — координати частинки;
- \(z\) — вимірювання;
- \(R\) — шум вимірювання.

---

## 7. Перевибірка частинок

Якщо ефективна кількість частинок зменшується, виконується Resampling для покращення стабільності фільтра.

---

## 8. Візуалізація

Програма відображає:

- зелену рамку детекції;
- жовту область пошуку;
- сині частинки;
- червону оцінку положення;
- траєкторію руху об'єкта;
- маску руху.
