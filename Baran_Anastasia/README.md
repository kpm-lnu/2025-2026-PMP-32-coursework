# Speech Emotion Recognition
Coursework project on speech emotion recognition using deep learning methods.

## Technologies
- Python
- PyTorch
- Librosa
- NumPy
- Matplotlib
- Scikit-learn

## Features
- Audio preprocessing
- Mel spectrogram extraction
- CNN model training
- Emotion classification
- Confusion matrix visualization
- Accuracy and loss graphs

## Dataset
RAVDESS emotional speech dataset.

## Project Structure

main.py — model training and evaluation  
sort_dataset.py — automatic dataset sorting  
dataset/ — prepared dataset folders

## Run

Install dependencies:

pip install torch librosa matplotlib scikit-learn numpy

Run project:

python main.py
